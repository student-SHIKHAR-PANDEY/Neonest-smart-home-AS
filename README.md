# ⚡ Smart Home Automation & Power Monitoring System

## 🚀 Overview

Enerlytics is a **next-generation IoT-based smart home automation system** built using ESP32, designed to deliver seamless appliance control and real-time energy monitoring.

The system integrates cloud connectivity via ESP RainMaker, enabling users to control and monitor devices remotely through mobile apps, voice assistants, and multiple control interfaces.

This project demonstrates a **scalable, intelligent, and real-world implementation of IoT + Embedded Systems + Cloud Integration**.

---

## ✨ Key Features

* 🔌 **Remote Appliance Control** via mobile app
* 📊 **Real-Time Power Monitoring** using ACS712 current sensor
* ☁️ **Cloud Integration** with ESP RainMaker
* 🗣️ **Voice Control Support** (Alexa & Google Assistant)
* 🎮 **Multi-Mode Control System**

  * Mobile App
  * IR Remote
  * Manual Switches
* ⚡ **Live Device Status Tracking**
* 🔄 **Customizable Appliance Configuration**

## 🛠️ Tech Stack

* **Hardware:** ESP32, ACS712 Current Sensor, Relays
* **Software:** Arduino IDE (Embedded C/C++)
* **Cloud:** ESP RainMaker
* **Integration:** Alexa, Google Assistant
* **Concepts:** IoT, Automation, Real-Time Monitoring

---

## ⚙️ System Architecture

1. ESP32 reads sensor data (current, device status)
2. Sends data to cloud via ESP RainMaker
3. User interacts via:

   * Mobile App
   * Voice Assistant
   * IR Remote / Switch
4. Devices are controlled in real-time with feedback
